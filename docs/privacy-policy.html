개인정보 처리방침

앱 이름: Project Tracker  |  시행일: 2026년 4월 19일  |  최종 수정일: 2026년 4월 19일

1. 수집하는 개인정보 항목

Project Tracker는 서비스 제공을 위해 다음과 같은 정보를 수집할 수 있습니다.

  • 이메일 주소 (선택): 다기기 동기화를 위해 계정 연동 시 수집
  • 앱 사용 데이터: 사용자가 직접 입력한 프로젝트 이름, 업데이트 내역, 보완 사항, 메모 등
  • 익명 사용자 식별자: 로그인 없이 사용 시 자동 생성되는 임시 ID (기기 내 저장)

2. 개인정보 수집 및 이용 목적

  • 프로젝트 데이터의 저장 및 동기화
  • 다기기 간 데이터 공유 서비스 제공
  • 서비스 품질 개선

3. 제3자 서비스 (Firebase)

본 앱은 Google의 Firebase 서비스를 사용합니다.

  • Firebase Authentication: 사용자 인증 (익명 및 이메일)
  • Firebase Firestore: 데이터 저장 및 동기화

Firebase의 개인정보 처리방침은 다음에서 확인하실 수 있습니다:
https://firebase.google.com/support/privacy

4. 개인정보 보유 및 이용 기간

  • 이메일 계정 사용자: 계정 삭제 요청 시까지
  • 익명 사용자: 앱 삭제 시 로컬 데이터 삭제, Firebase 서버 데이터는 90일 후 자동 삭제

5. 개인정보의 제3자 제공

본 앱은 수집한 개인정보를 외부에 판매하거나 제3자에게 제공하지 않습니다. 단, Firebase(Google)는 서비스 운영을 위한 처리자로서 데이터를 취급합니다.

6. 사용자의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 저장된 데이터 열람 및 삭제 요청
  • 계정 삭제 요청
  • 데이터 처리에 대한 동의 철회

요청 사항은 아래 이메일로 연락해 주세요.

7. 어린이 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 의도적으로 미성년자의 개인정보를 수집하지 않습니다.

8. 개인정보 처리방침 변경

본 방침은 법령 또는 서비스 변경에 따라 업데이트될 수 있으며, 변경 시 본 페이지를 통해 공지합니다.
